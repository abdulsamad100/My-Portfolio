## 🚀 Getting Started

To view the project locally:

1. Clone the repository:

   \`\`\`bash
   git clone https://github.com/abdulsamad100/My-Portfolio.git
   \`\`\`

2. Navigate to the project directory:

   \`\`\`bash
   cd My-Portfolio
   \`\`\`

3. Open \`index.html\` in your preferred browser.
